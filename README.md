# SpringBootAngular

E-Commerce Application, provides below services for Admin & Customer CRUD operations 

Services :


- Admin 

Adding products

Updating products

Deleting products

Manage Orders


- Customer 

Registering into System

Login into Website

Updating Address

Adding product to Cart

Updating/ Deleting the Product

Placing the order


- Technologies :

Java 16

Spring Boot

Hibernate with JPA

MySQL

PostMan (For testing the web services)
